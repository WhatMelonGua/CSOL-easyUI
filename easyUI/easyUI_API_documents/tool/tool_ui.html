工具表 tool_ui

提供了 UI设计可能使用的 一系列函数。

Functions

.bindPosition (user,target,offset) [·] 将user的位置 偏移offset移动到 target的位置 并进行绘制
.bindScale (user,target,offset) [·] 将user的缩放 偏移offset同步至 target的缩放 并进行绘制触发 Lua的 GC垃圾回收
.setsCommand (sets, cmd) [·] 将目标sets表的 x/y/width/height/r/g/b/a 全部设定为 cmd中对应属性的值
cmd中没有的属性则 继承原值不变
.setsColorSet (sets, color) [·] 将目标sets表的所有setArg 都变成 color颜色
.oppoSet (table) [·] 生成 目标setArg/vector/color 的数字全部取 相反数 的表
.recipSet (table) [·] 生成 目标setArg/vector/color 的数字全部取 倒数 的表
.vectorAdd (vectorA, vectorB) [·] 求算 vectorA 和 vectorB 两个向量相 加和
.vectorMultiply (vectorA, vectorB) [·] 求算 vectorA 和 vectorB 两个向量相 乘积
.vectorLength (vector) [·] 求算 目标vector 的长度
.rgbAdd (rgbA, rgbB) [·] 求算 rgbA 和 rgbB 两个向量相 加和
.rgbMultiply (rgbA, rgbB) [·] 求算 rgbA 和 rgbB 两个向量相 乘积

Functions

.bindPosition (user,target,offset)
将 user对象 的位置 改变到 target对象 的位置 偏移offset后的地方
注意，此函数执行完毕后将自动执行 绘制函数更新屏幕

Parameters:

  • user uiComponent | UI.Box | UI.Text 需要改变位置的组件，支持 uiComp/原生组件 均可
  • target uiComponent | UI.Box | UI.Text 需要跟随的目标，支持 uiComp/原生组件 均可
  • 可选 offset vector 相对目标位置的 偏移距离。默认为{x=0,y=0} 代表不偏移。
.bindScale (user,target,offset)
将 user对象 缩放为 target对象 的缩放值 * 乘偏移offset 后的大小
此函数只支持 uiComponent！
注意，此函数执行完毕后将自动执行 绘制函数更新屏幕

Parameters:

  • user uiComponent 需要改变缩放的组件，仅支持 uiComp
  • target uiComponent 需要跟随的目标，仅支持 uiComp
  • 可选 offset vector 相对目标缩放值的 缩放。默认为{x=1,y=1} 代表不偏移。
.setsCommand (sets,cmd)
将 sets表 内所有 setArgs 的各项属性，变为 cmd表 内对应属性的值。

Parameters:

  • sets table 包含一系列setArg的表，如class_uiComponent.sets 表
  • cmd table cmd表 包含setArg的任意属性即可，不需要全部填写
    如 {x=1} | {r=244,g=255} | {x=0,y=0,width=20,height=20,r=255,g=255,b=255,a=200}
    都是符合规范的传入

Usage:

sets = {
{x=0,y=0,width=20,height=20,r=255,g=25,b=20,a=200},
{x=0,y=30,width=40,height=40,r=55,g=85,b=200,a=255},
{x=20,y=0,width=60,height=20,r=155,g=125,b=120,a=0}
}
cmd = {x=0,r=255,g=255,b=255}
tool_ui.setsCommand(sets,cmd)
执行以上，sets将变为： 
sets = {
{x=0,y=0,width=20,height=20,r=255,g=255,b=255,a=200},
{x=0,y=30,width=40,height=40,r=255,g=255,b=255,a=255},
{x=0,y=0,width=60,height=20,r=255,g=255,b=255,a=0}
}
sets表的子元素 将尽量同步至 cmd表 所具有的属性，这一般用来批量更改某组 setArgs 的颜色。
这在更改 文本显示 / icon图标[一般图标都是一种颜色]模板 会很有用。
.setsColorSet (sets,color)
将sets表内所有 setArgs变为对应的 color颜色。不影响原先的x、y、widht、height

Parameters:

  • color color 颜色，rgba都不可省略。
.oppoSet (table)
将对应 table的所有 数字值取反，并返回新表。

Parameters:

  • table vector 一般希望您传入vector，实际上只要保证 table 的所有元素都是可计算的数字即可。

Return:

  • table vector 与原表数字相反(取负数)的 新表，key键与原表对应。
.recipSet (table)
将对应 table的所有 数字值取倒数，并返回新表。

Parameters:

  • table vector 一般希望您传入vector，实际上只要保证 table 的所有元素都是可计算的数字即可。

Return:

  • table vector 与原表数字 取倒数 的 新表，key键与原表对应。
.vectorAdd (vectorA, vectorB)

Usage:

vectorA={x=5,y=10}
vectorB={x=-2,y=3}
result = tool_ui.vectorAdd(vectorA, vectorB)
-- 返回结果 result = {x=3,y=13}
.vectorMultiply (vectorA, vectorB)

Usage:

vectorA={x=5,y=10}
vectorB={x=-2,y=3}
result = tool_ui.vectorMultiply(vectorA, vectorB)
-- 返回结果 result = {x=-10,y=30}
.vectorLength (vector)

Usage:

vector={x=3,y=4}
result = tool_ui.vectorLength(vector)
-- 返回结果 result = 5 = (3^2+4^2)^0.5
.rgbAdd (rgbA, rgbB)
效果类似vectorAdd，传入值为 color类型
.rgbMultiply (rgbA, rgbB)
效果类似 vectorMultiply， 传入值为 color类型
