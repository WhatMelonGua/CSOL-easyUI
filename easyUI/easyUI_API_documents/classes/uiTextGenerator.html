伪类 class_uiTextGenerator

相比于 uiComponent 和 uiAnime 两伪类

，uiTextGenerator 更像是针对 uiComponent的 处理工具。
它是一个 类似文字发生器 的对象，用于弥补 csol官方 禁止UI.Text 文字类的不足。
用于 从字库取出文字setArgs -> 文字发生处理 -> 赋给uiComp对象

目前的字库提供了 [26英文字母大小写] + [0~9数字] + [正常的英文符号] +
[正常的中文符号] + [最常用的3000汉字]

若您还希望添加一些 ♥/♦ 等特殊符号，甚至表情😀
欢迎使用 uiComponent.fromTemplate() 基于 UI.BoxPainter 定制添加
(这并不能添加至字库，仅是能提供绘制)
后期会提供 sets表 转入 字符库 的插件

特别感谢：

字库基于 LanaPixel像素体 生成，作者使用的下载源为 猫啃网免费字体
感谢字体作者的贡献、开源。

名词解释：

-- 可选参数：您可以 省略不写 此参数，代码自动取默认值；填写则按照您的输入值为参数 通常其在 Functions 的参数中可以找到，形式为:

  • 可选 parameter T : xxx，默认值为val

    • Attributes

    ui_type (global) 标识了此伪类的属别，可能仅在debug时使用。
    其值为'uiTextGenerator' (string) 不可更改
    font 该 文字发生器使用的 字库
    easyUI.lua采用 LanaPixel像素体 作为标准字库。

    Functions

    .new (uiFont) [·] 新建 文字发生器，并设定使用 uiFont的传入值 作为字库
    :textComponent (pos,text,fontSize,color,space,fixAble) 创建一个 uiComp对象 生成文字画面。很多参数都是可选的
    :textUpdate (comp,textSet) 将 comp对象(uiComponent组件) 变为新传入的 textSet 字符展示。
    :fontSizeSet (comp,fontSize) 改变传入 comp对象(uiComponent组件) 的字体字号为 fontSize。
    :_getTextSets (str, space, color) 内置函数，不供给用户使用，用于从字库获取当前 字符串的信息并转换为一系列 setArgs
    ._str2table (str) [·] 内置函数，用于将字符串str 转换为 table类型 的表格
    ._getBoxNum (fset) [·] 内置函数，根据字库提供的 字体信息fset 计算该字 需多少UI.Box
    ._setTransform (fset, color, pointer, gap) [·] 内置函数，不供给用户使用。将字体信息 根据缩进、字位等 转换为uiComp的 setArg

    Parameters

    textSet 当您更新uiComp的 显示文字时，需要使用到 此参数结构。
    uiFont 字库，一般采用 LanaPixel字库
    vector 表示二维向量的参数结构
    color 表示颜色的参数结构

    Attributes

    ui_type
    标示着该对象在easyUI的类型为uiTextGenerator

    String:

    font
    文字发生器 所使用的 字库，目前只有LanePixel字库适配。 其变量名称即为LanePixel。
    您或许可在附赠文件的 ('LanaPixel_xxx.lua') 文件中 对该字库一睹芳容

    uiFonts:

    • uiFont "uiFont" easyUI.lua 可使用的 规范字库。

    Functions

    .new (uiFont)
    创建一个uiTextGenerator实例，一般全局您只用创建 一次/一个文字生成器 即可满足。

    Parameters:

    • uiFont uiFont 使用的字库，需要您传入，一般填写 LanaPixel 即可

    Return:

    • uiTextGenerator uiTextGenerator 文字发生器 对象

    Usage:

    tGenerator = class_uiTextGenerator.new(LanaPixel)
    :textComponent (pos,text,fontSize,color,space,fixAble)
    不要被众多的参数 吓到，很多都是可 不填写/选择 的！
    基于 文字发生器 根据字库创建 uiComponent对象
    返回的 uiComp对象将 按照您的设定显示文字。(使用:Draw()函数 )
    该返回对象支持所有的 uiComp方法。

    Parameters:

    • pos vector 文字的显示位置
    • text string 需要显示的文本内容
    • 可选 fontSize number 字号大小(即占地 nxn 像素)，根据字库进行 uiComp的缩放。
      默认值为 字库的基准大小。[代码使用的LanaPixel字库为 10像素 生成的]

    • 可选 color color 文本的 色彩 (rgba)。默认为黑色，即 {r=0,g=0,b=0,a=255}
    • 可选 space vector | number 用于设定字间距 和 行间距(对应 x | y 因子)，单位：像素。
      当传入值为 单个number 时 代表字间距，行间距取默认值 2
      为 vector(table) 时，默认值为 {x=1, y=2}
      设定间距时，其值是相对 字库基准大小 的，不需要根据 fontSize 大小变化。

    • 可选 fixAble bool 该uiComp是否应用屏幕修正，默认为 true ，应用。

    Return:

    • uiComponent uiComponent 已经设定好文字setArgs的 uiComponent对象
    :textUpdate (comp, textSet)
    将 comp组件 的显示更新为 textSet设定的文本内容

    Parameters:

    • comp uiComponent 需要进行更新的 uiComponent对象
    • textSet textSet 变更文本的设定。
      参数例子: {text='显示的文本内容', space={x=1,y=12}, color={r=0,g=0,b=0,a=255}}
    :fontSizeSet (comp, fontSize)
    将 comp组件 的字号更新为 fontSize

    Parameters:

    • comp uiComponent 需要进行更新的 uiComponent对象
    • fontSize number 要设定的 字号 (单位:像素)
    :_getTextSets (str, space, color)
    内置函数，不供给用户使用，用于从字库获取当前 字符串的信息并转换为一系列 setArgs

    Parameters:

    • str string 文本内容
    • space vector 字/行间距
    • color color 色彩

    Return:

    • sets table 可以使 uiComp使用 :setLink()链接 的 setArgs表。
    ._str2table (str)
    内置函数，用于将字符串str 转换为 table类型 的表格

    Parameters:

    • str string 文本内容

    Return:

    • strTable table 字符串 拆分为单个字符的 表
    ._getBoxNum (fset)
    内置函数，根据字库提供的 字体信息fset 计算该字 需多少UI.Box

    Parameters:

    • fset table 字库对应该字符的 解释信息

    Return:

    • number number 该字符需要使用的 UI.Box 数量
    ._setTransform (fset, color, pointer, gap)
    将字库 某字符的解释信息(fset) 按设定转换为 setArg。
    注意：一个字符含多个setArg (即多个Box才能构成一个字)，此以pointer指针指定所需的Box次序。

    Parameters:

    • fset table 字库对应该字符的 解释信息
    • color color 生成setArg的 色彩
    • pointer number 指针，指至需要的 Box的SetArg 在组成该字的所有Box间的位次。
    • gap vector 字/行间距导致的偏移值

    Return:

    • set setArg 官方标准的 setArg数据信息

    Parameters

    textSet
    使用 :textUpdate(textSet) 函数所需的输入参数，为 table

    Fields:

    • text string 文本内容
    • 可选 space vector | number 用于设定字间距 和 行间距(对应 x | y 因子)，单位：像素。
      当传入值为 单个number 时 ，代表字间距，行间距取默认值 2
      当传入值为 vector(table) 时，如{x=2,y=2}中 x代表字间距，y代表行间距
      设定间距时，其值是相对 字库基准大小 的，不需要根据 fontSize 大小变化。

    • color color 文本的显示色彩
    uiFont
    easyUI.lua的字库 伪类，因只需要指定不需要使用，不在此文档详细说明介绍。
    下方的内容 仅做了解即可 其也有ui_type属性，且值为 'uiFont'
    标准字库为 LanaPixel (lua中的变量名称也如此)

    字库的单个字符 解释信息为:
    每4个数字 代表1个UI.Box的 x、y、width、height。若表长>4*int，最后一位代表该字符的 宽度

    Fields:

    • :_getFset(char) function 获取对应字符 char在表内的 解释信息table
    • '[Char]' string 单个字符key，对应的value是 该字符的解释信息
    vector
    二维向量类型，可以表示 位置、移动量、横纵因子等。视情况而定。

    Fields:

    • x int 水平方向(x轴)的因子
    • y int 竖直方向(y轴)的因子
    color
    表示颜色的参数结构

    Fields:

    • r int red 色(0~255)
    • g int green 色(0~255)
    • b int blue 色(0~255)
    • a int alpha 透明度 (0~255)
