easyUI.Lua

easyUI.lua 是根据CSOL缔造者"Lua studio"常见的UI.Box绘制需求而进行设计的第三方工具库。

其目的在于构建组件化的UI.Box进行多Box组件的管理，而尽可能减少对单个Box进行操作；

文档中可能存在诸多函数，但实际应用中您可能仅需要掌握其中几个关键函数
更多的其他函数是为了拓展性考虑，因此不必担心。

本工具库主打UI.Box的管理绘制，并不提供多余的功能。概览如下：

1、多个Box可实现打组，打组后(组件化)可进行克隆、移动、组件中心设定、90°旋转、镜像等常规操作

2、对于灵活的绘制功能，工具库提供了回调函数、绘制插件函数等，让UI.Box的绘制更为灵活

3、支持对组件添加动画效果，并提供了当今常见的缓动函数帮助您设计更加平缓柔美的动画效果

4、组件可生成文字[代替UI.Text]，现有字库中已包含 英文字母、常见符号、约达3100的常用汉字


导入方法 (project.json文件) 
{
    "game": ["game.lua"], 
    "ui": ["LanaPixel_x.lua","template.lua","easyUI.lua","ui.lua"]
}
-- 解释:
-- LanaPixel_x 代表 字库lua文件
-- template 是可能会使用的 模板文件( UI.Box Painter 导出)
-- easyUI 是工具库本体

其提供了以下三大伪类来实现UI.Box组件化管理

伪类 (class_)

class_uiComponent 此 伪类 提供了最为基准的UI.Box组件化，并包含了组件的一系列图形学变换作方法。 是easyUI 库通用的 基本伪类
class_uiTextGenerator 此 伪类 作为 class_uiComponent 文字生成器使用，可根据 字库 生成显示文字的uiComponent，或对已存在的uiComponent进行文本替换等。
class_uiAnime 此 伪类 为针对 class_uiComponent 的动画序列器，可生成动画对象，完成辅助的组件动画效果 例如运动、缩放、色彩更替等

工具 (tool_)

tool_table 包含了对表 深层清空、逐层输出的基本操作，维持easyUI 库的正常运行所必须，当然您也可于您自己的代码中利用
tool_ui 包含ui设计可能需求的其他函数，提供了原生 UI.Box | UI.Text[数字] 对uiComponent进行位置跟踪的兼容方案

全局变量 (GLOBAL_)

GLOBAL_WINSIZE 只读，不可修改。
当前 客户端(玩家电脑) 的 屏幕尺寸
GLOBAL_CODETIME 只读，不可修改。
代码载入完毕的 初始时间 (也可以理解为玩家开始游玩的时间节点)

什么是 伪类 (拓展介绍)

伪类是通过Lua语言的 表( table ) 来完成实现的，通过元表进行链接。 经常使用的语句是 setmetatable(target, class) 来完成将 class 类附加于 target 表的操作 对于class 应该有 class.__index = class 使得子表可以通过key键值对继承到class的 方法/属性 实现类似 某些计算机语言 中 类 的概念，所以称之为 伪类

伪类可以帮助 作者更好的安排编程对象、理清思路，可于下方 菜鸟教程 对元表的介绍中对此方法进一步理解
